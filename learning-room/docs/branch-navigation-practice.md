# Day 1 Practice Exercise: Branch Navigation

**This file only exists on the `day1-practice` branch.**

Use this exercise to practice navigating between branches on GitHub and Git.

## Challenge: Find the Differences

Your task is to find all the differences between the `main` branch and the `day1-practice` branch.

### How to Complete This Challenge

1. **View this file on `day1-practice` branch** - you are here now
2. **Switch to the `main` branch** in GitHub
   - Click the branch selector dropdown (usually shows "main")
   - Select "day1-practice" from the list
   - Click "Go to file" or navigate to this directory
3. **Compare the files** between branches
4. **Make a note** of what changed:
   - What files are different?
   - What lines were added or removed?
   - Can you read the changes with your screen reader?

## Practice Reading Diffs

Once you've found the differences, practice reading the unified diff format:

- **Deleted lines** start with a `-` (minus sign)
- **Added lines** start with a `+` (plus sign)
- **Context lines** (unchanged) have a space at the start

This is how you'll read pull requests!

## Submit Your Findings

When you're ready, open an issue on the main repository and describe:
- What files are different between branches
- What content changed in each file
- One thing you learned from comparing these branches
